### Decisions made
See `workstreams/main-decisions.md` (2026-07-09 section) + the impl-notes file. Highlights + WHY:
- **cycle_id uses ticker (CUSIP fallback), not CUSIP** — operator wanted readable ids (`CIEN_COMMON_2`).
- **classifications keyed by ticker, PRIMARY over EODHD** — the EODHD key returns HTTP 403 on
  fundamentals (separate plan tier); the manual file sidesteps that without a plan upgrade.
- **SMH secondary benchmark, NO fatal preflight (SD-SMH-1)** — SPY is required (fatal preflight); SMH
  missing → null trio per task spec, so it must never abort the run.
- **build_views may now fetch** (fundamentals) — deliberate relaxation of its no-network contract.
